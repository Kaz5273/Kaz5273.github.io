Projet Mots & Merveilles Online

  • Information du projet:
  • Catégorie:Application web
  • Langages:PHP, Bootstrap, CSS, JavaScript

Resumé du besoin:

Créez une application web pour l'association afin qu'elle puisse gérer ses adhérents, qu'ils soient apprenants ou bénévoles.

Objectif:

Développement de nouvelles fonctionnalités dans l'application, réalisation de tests pour garantir la qualité des livrables.


Prestation livrée:

Compétence acquise:

  • Utilisation de Symfony PHP

  • Création de formulaire

  • Création des User Storie

  • Traiter les demandes attendus

  • Mise en place des rôles/sécuriser les routes

  • Récupération de mot de passe via l'Email
